Variants

INVALID_HANDLE

Invalid HELEMENT.

PASSIVE_HANDLE

Attempt to use HELEMENT which is not attached to document.

INVALID_PARAMETER

Parameter is invalid, e.g. pointer is null.

OPERATION_FAILED

Operation failed, e.g. invalid html passed.

OK_NOT_HANDLED
